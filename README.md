# A Pokemon interface built with React

Search for common Pokemon in this React App. Features infinite scrolling, filter search as you type, and detailed information about each Pokemon on click.

This project was bootstrapped with [Create React App](https://github.com/facebookincubator/create-react-app).

## Local Installation

In your terminal:

  1. Git clone https://github.com/speterlin/react-pokemon.git
  1. cd react-pokemon
  1. npm install
  1. npm start

Then navigate to [http://localhost:3000](http://localhost:3000)
